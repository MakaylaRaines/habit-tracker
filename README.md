# HabitTrack

## Project Description
HabitTrack is a simple web application that helps users track daily habits such as exercising, studying, or drinking water. Users can add habits, mark them as completed, and remove habits they no longer want to track.

## Problem Statement
Many people struggle to stay consistent with daily habits. This application provides a simple interface to help users stay organized and maintain routines.

## Technologies Used
Frontend: HTML, CSS, JavaScript  
Backend: Node.js with Express  
Data Storage: JSON file  
Version Control: GitHub  

## Setup Instructions
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
4. Open `http://localhost:3000` in a browser

## Design Patterns
MVC Pattern: Separates data, logic, and user interface.  
Factory Pattern: Used when creating new habit objects.
